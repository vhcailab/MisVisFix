API Key Setup Guide

Step Instructions Visual Guide
OpenAI Key Setup
  1. Go to the API Key Page: Visit https://platform.openai.com/account/api-keys and click “+ Create new secret key.”
  2. Fill in Key Details: (Optional) Add a name, choose your project and permissions, then click “Create secret key.”
  3. Copy and Save the Key: Click “Copy” to save your key somewhere secure. You won’t be able to view it again.
Claude Key Setup
  1. Go to the API Key Page: Visit https://console.anthropic.com/settings/keys and click “+ Create Key.”
  2. Set Workspace and Name: Select your workspace from the dropdown, enter a name for the key, and click “Add.”
  3. Copy and Save the Key: Click “Copy Key” and store it safely. You won’t be able to view it again after closing.
Assistant ID Setup
  1. Go to the Assistants Page: Visit https://platform.openai.com/assistants and click “+ Create” to add a new assistant.
  2. Configure the Assistant: Enter a name, model, and then save the assistant.
  3. Copy the Assistant ID: Once created, click your assistant and copy the ID shown in the detail panel.
